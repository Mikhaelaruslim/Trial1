Metadata Lab — Student Template

These tasks mirror what web professionals do to make sites discoverable, accessible, and visually effective. Each step builds a skill you’ll use in real projects.

Step 1: Metadata (Real-World Purpose)

  • [ ] Add <meta charset="utf-8">
  • [ ] Brainstorm 3 titles (comments), then choose 1 and put it in <title>
  • [ ] Write 2 description drafts (comments), then choose 1 and add <meta name="description">
  • [ ] Verify in the browser + View Source

Quick Examples

<meta charset="utf-8">
<title>My First Metadata Page</title>
<meta name="description" content="A short, friendly summary of this page (about 50–160 characters).">
Why? These lines are what browsers, search engines, and social media use to display and rank your site. Getting them right is a core web skill.
